# 날아다니는 파스타 몬스터 (날파몬)

크툴루 신화를 바탕으로 한 추리+스릴러 횡스크롤 게임.  
Night in the Woods처럼 2D 횡스크롤 이동, 역전재판/단간론파 식 대화 시스템.

## 특징
- 2D 캐릭터 이동 + 상호작용
- 각종 미니게임
- Yarn Spinner(또는 기타 대화 툴) 기반 분기 대화
